Detailed Description

Represents an ARP protocol layer. Currently only IPv4 ARP messages are supported

void ArpLayer::computeCalculateFields ( )
virtual

Calculate the following fields:

  • arphdr::hardwareType = Ethernet (1)
  • arphdr::hardwareSize = 6
  • arphdr::protocolType = ETHERTYPE_IP (assume IPv4 over ARP)
  • arphdr::protocolSize = 4 (assume IPv4 over ARP)
  • if it's an ARP requst: arphdr::targetMacAddr = MacAddress("00:00:00:00:00:00")
