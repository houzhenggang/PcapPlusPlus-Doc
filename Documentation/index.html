Introduction

PcapPlusPlus is a multiplatform C++ network sniffing and packet parsing and manipulation framework.
PcapPlusPlus is meant to be lightweight, efficient and easy to use.

Installation

Please visit the PcapPlusPlus web-site in order to see download and installation instructions.

Getting Started

Packet capturing engines (Pcap++):

  • PcapLiveDevice, PcapLiveDevice.h - wrapper for libPacp engine (Linux)
  • WinPcapLiveDevice, WinPcapLiveDevice.h - wrapper for WinPcap engine (Windows)
  • PcapRemoteDevice, PcapRemoteDevice.h - remote live device, supported by WinPcap(Windows)
  • PfRingDevice, PfRingDevice.h - Vanilla PF_RING device (Linux)
  • PcapFilter.h - structured filter engine

Pcap file wrappers (Pcap++):

  • PcapFileReaderDevice, PcapFileDevice.h - read Pcap files
  • PcapFileWriterDevice, PcapFileDevice.h - write Pcap files

Packet parsing and manipulation (Packet++):

  • RawPacket, RawPacket.h - raw data class
  • Packet, Packet.h - parsed packet class
  • Layer, Layer.h - protocol base class

Supported protocols (Packet++):

  • EthLayer, EthLayer.h - Ethernet protocol
  • ArpLayer, ArpLayer.h - ARP protocol
  • VlanLayer, VlanLayer.h - VLAN protocol
  • IPv4Layer, IPv4Layer.h - IPv4 protocol
  • IPv6Layer, IPv6Layer.h - IPv6 protocol
  • TcpLayer, TcpLayer.h - TCP protocol
  • UdpLayer, UdpLayer.h - UDP protocol
  • HttpRequestLayer, HttpLayer.h - HTTP request protocol
  • HttpResponseLayer, HttpLayer.h - HTTP response protocol
  • PayloadLayer, PayloadLayer.h - unknown layer or payload

Common utilities and infrastructure (Common++):

  • IPv4Address, IPv6Address, IpAddress.h - IP address wrapper classes
  • MacAddress, MacAddress.h - MAC address wrapper class
  • LoggerPP, Logger.h - PcapPlusPlus logger
  • IpUtils.h - all kinds of network helpful utilities
